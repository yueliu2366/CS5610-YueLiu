Experiment1- AngularJs-SimpleString

This is my first try out of AngularJS. In this experiment, I define a string {{hello}} in html page. Then I define a controller in js file. The UI and js file communicate with each other via a built-in variable called $scope.

This experiment is based on the Experiment1- AngularJs-SimpleString. The ng-controller is still HelloWorldController. However I try to pass an object "employees" via ng-repeat. It is more common to pass object instead of just passing a string.

Experiment4-Add a Favorite movie

This experiment is also based on what Prof. Jose did on class. I implement a search bar firstly. The beauty of techonic behind this search bar is the ng-model and the ng-click fields. By defining these two attributes, I am able to bind a variable and a function in html to the corresponding variable and function in js file. Besides, adding a favorite movie is the same as adding a movie which I achieved in previous experiment.

Experimen5-Show details

Based on Experiment4-Add a Favorite movie, I add a button which enables users to see more information about some particular movies. When clicked, the button will invoke a function called "detailsMovie" in js file, this function will fetch detail information via idIMDB and represent to html through $scope.details
