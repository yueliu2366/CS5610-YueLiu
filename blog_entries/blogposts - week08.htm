Week08 Experiment

Experiment1-JQuery- SlideDown

In this experiment, I try out jQuery slideDown() method. The jQuery slideDown() method is used to slide down an element. The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds. The optional callback parameter is a function to be executed after the sliding completes.

Experiment2-JQuery- SlideUp

In this experiment, I try out jQuery slideToggle() method. The jQuery slideUp() method is used to slide up an element. The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds. The optional callback parameter is a function to be executed after the sliding completes.

Experiment3-JQuery- Toggle

In this experiment, I try out jQuery slideToggle() method. The method toggles between the slideDown() and slideUp() methods. If the elements have been slid down, slideToggle() will slide them up. If the elements have been slid up, slideToggle() will slide them down.
