Login Module

Description:

The login page is based on Bootstrap,The following are id and class which is used in the UI design of login page: login-overlay, modal-dialog, modal-content, modal-header,modal-body. The main layout is built on gridlayout of Bootstrap .The panel, which is a row class, is divided into two equal length part. The left side is for login and the right side provides button and link to registration and additional information.
