Week03-Experiment01

Experiment Description:

In this experiment, I try out two techniques: 1. Dynamic title 2. Dynamic Array.

Here is the sinppet of JS file:


$(function () {
    var mainTitle = "This is a dynamic title";
    $("#mainTitle").html(mainTitle);
    var genres = ["Horror", "Comedy", "Drama"];
    var ul = $("ul#genres");
    ul.empty();
    for (var i in genres) {
        var li = $("<li>" + genres[i] + "</li>");
        ul.append(li);
    }
});

Both title and array will be modified dynamically after the html file is load. JQuery will bind with variable ul, then we can manipulate this object model in the JS file.
