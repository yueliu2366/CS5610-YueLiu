Week03-Experiment03

Experiment Description:

In this experiment, I try the following in JQuery UI:

-drag

-reorder

-resize

These techniques can be used to design responsive page. Imaging we are developing a personal profile page, it is very common to add a field to describe skills that users have, and we also want these skills to be reorder in o rder to emphasize the top skill.

Here is the sinppet of JS file:


$(function () {
    $(".reorderme").sortable({
        axis: "y"
    });
    $(".dragme").draggable({
        grid: [20, 20],
        start: function () { console.log("start drag"); },
        stop: function () { console.log("stop drag"); }
    });
    $(".resizeme").resizable({
        handles: "n, e, s, w, ne, se, sw, nw"
    });
});
