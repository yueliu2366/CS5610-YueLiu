Experiment Description:

In this experiment, I build a prototype of single page website.

Here is the sinppet of JS file:


var app = angular.module("MovieApp", ['ngRoute']);
app.config(['$routeProvider',
function ($routeProvider) {
    $routeProvider.
    when('/home', {
        templateUrl: 'movie/home.html'
        // ,controller: 'PhoneListCtrl'
    }).
    when('/profile', {
        templateUrl: 'movie/profile.html'
    }).
    when('/details', {
        templateUrl: 'movie/details.html'
    }).
    otherwise({
        redirectTo: '/home'
    });
}]);

Notice thatngRoute should be added to angular.module() The rest parts are trivial.
