Experiment Description:

In this experiment, I try out jQuery slideToggle() method. The jQuery slideUp() method is used to slide up an element. The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds. The optional callback parameter is a function to be executed after the sliding completes.

Here is the sinppet of JS file:


$("#flip").click(function () {
    $("#panel").slideUp();
});

Here is the sinppet of CSS file:


#panel, #flip {
    padding: 5px;
    text-align: center;
    background-color: #e5eecc;
    border: solid 1px #c3c3c3;
}
#panel {
    padding: 50px;
    display: none;
}
