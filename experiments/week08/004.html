Experiment Description:

In this experiment, I try out jQuery Animations - The animate() Method By default, all HTML elements have a static position, and cannot be moved. To manipulate the position, remember to first set the CSS position property of the element to relative, fixed, or absolute!

Here is the sinppet of JS file:


$(document).ready(function(){
    $("button").click(function(){
        $("div").animate({left: '250px'});
    });
});
